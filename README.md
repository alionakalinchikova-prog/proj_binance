# Курс криптовалют к доллару(через Binance)

Простая консольная программа на C# для получения курса криптовалют с биржи Binance.

## Возможности

- Получение текущего курса BTC, ETH, BNB и других монет
- Сохранение результатов в JSON-файл с отметкой времени
